Overview

The dashboard lists every server with live status, player count, and CPU/RAM use. Click a server to open its detail page — terminal, files, config, backups, schedules, and the custom tabs below.

  • Use the sidebar to jump between the dashboard, Discover, and this documentation.
  • Right-click a server card for quick actions (start/stop, clone, delete).
  • Navigation is instant — pages swap without a full reload, and the live status keeps updating.

Managing servers

Open a server, then use the Terminal tab to watch the console and send commands. The control buttons handle the lifecycle:

  • Start / Stop / Restart — graceful start and shutdown. A start always clears any stuck/orphaned process first, so it never fails on a locked world file.
  • Kill — force-stops a frozen server.
  • Config — RAM (min/max), the server port, Java version, and startup flags. Recommended RAM is suggested for your machine, and the port is checked for conflicts.
  • Files — browse and edit any server file in-place.

Version upgrades

On a server's Version Updater tab, the 1-Click Version Upgrade card shows your current version and any newer ones. Pick one and hit upgrade.

  • Your progress is preserved. Worlds, player inventories, ender chests, stats, and advancements are snapshotted, the jar is swapped, then every world is verified afterwards and restored if anything is missing.
  • A full backup is taken automatically before the swap.
  • The server restarts itself if it was running. Supports vanilla, Paper, Purpur, Folia, and Fabric.

Discover mods & packs

The Discover hub searches Modrinth for modpacks, mods, datapacks, resource packs, and shaders.

  • Import a modpack — paste a Modrinth or CurseForge link, or upload a .mrpack / CurseForge .zip. The loader and Minecraft version are detected, the server is created, and every mod is downloaded automatically.
  • Install onto a server — versions are auto-filtered to that server's loader and Minecraft version, and required dependencies install automatically.
  • Use Back / Next to page through results.

World Tools

The World Tools tab regenerates a single dimension for fresh terrain and resources — ideal for periodic Nether/End resets.

  • Reset Overworld / Nether / End — only that dimension's terrain is wiped and regenerated on next load.
  • Player inventories, ender chests, stats, and the other dimensions are kept.
  • A full backup is taken first, and the server restarts itself if it was running.

Live Map

The Live Map tab shows a live 3D render of your actual world, powered by BlueMap.

  • Real terrain and builds, with coordinates and online player positions updating in real time.
  • Private to this dashboard — it's served behind your login and never exposed publicly.
  • The map fills in as it renders in the background after first install.

Backups

The Backup tab creates and restores full server backups.

  • Set how many backups to keep and which folders to exclude.
  • A backup is taken automatically before risky actions (version upgrades, world resets).
  • Restore returns the server to any saved point.

Players & access

  • Player Controls — whitelist players, op them, and ban known players from a dropdown or by name.
  • Public access — your server is reachable at its public address through a self-healing tunnel that reconnects and holds the same port automatically.

Performance

To keep a server smooth and light on RAM:

  • Use tuned Aikar's JVM flags (G1GC) instead of plain -Xms/-Xmx — far fewer lag spikes.
  • Lower view-distance and simulation-distance in Paper config for the biggest RAM/CPU win.
  • Cap entities and mobs per chunk, and let unused chunks unload.
  • Match max RAM to your players — more isn't always better; oversized heaps make GC pauses worse.
